- The repository intentionally excludes `AGENTS.md` and built-in prompt templates; do not add them unless that distribution rule changes.
- Installing or deleting skills under `$CODEX_HOME/skills` still requires a Codex restart before runtime discovery is guaranteed.
- External or newly added skills must pass lifecycle and poisoning checks before default routing.
- Lifecycle scanner allowlist entries must remain exact-match with rationale; do not use broad suppressions.
